# MEAN app example
An example of MEAN (MongoDB, Express, Angular, Node.js) app with Socket.io and Material Design
Features includes:
* Chats
    * Direct Messages
    * Chat Rooms
* Basic store management
    * List products
    * Filter products
    * Add product to basket
    * View basket
    * Order products
    * View all orders

# Install
#### Build and run server
```
npm i
tsc
node bin/www
```
#### Build angular
```
cd angular-app
npm i
ng b
```
